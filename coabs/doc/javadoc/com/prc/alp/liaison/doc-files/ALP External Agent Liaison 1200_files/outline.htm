Interactor
An interface for an object that knows how to perform a specific kind of interaction with an external agent
Example: CoABSLookupInteractor
Generic action methods
begin(), send(), receive(), normalEnd(), abnormalEnd()
State represented by arbitrary object
Each action consumes and produces a state
nextAction()
Executes in context of an instance of an interaction thread
But can be multi-threaded
Interaction
An instance of a dynamic, bounded sequence of liaison actions by an interactor
Each action of the interactor produces a new state
Start with begin()
nextAction() picks next interactor action to perform
Continue until normalEnd() or abnormalEnd()
Actions typically produce and save some intermediate and final results with a LiaisonDeputy
Can be started synchronously (wait for completion) or asynchronously (poll for completion)

Code Example
// Setup liaison with CoABS Grid
LiaisonManager mgr = createLiaisonManager();
CoABSLiaisonDeputy dep;
dep = (CoABSLiaisonDeputy) mgr.createLiaisonDeputy(LConstants.COABS_SOCIETY,
                                                   getDelegate(),
                                                   getParameters());
mgr.start(true);
dep.start(“ALP_Society”, null, true);
// Register with the Grid
CoABSRegistryInteractor reg = new CoABSRegistryInteractor(dep);
Interaction regInt = new Interaction(dep, reg, dep.externalManager());
regInt.start(true);
// Find the RETSINA forecast agent
CoABSAgentDescription desc = new CoABSAgentDescription();
desc.name = “CNNForecastWeatherAgent”;
CoABSLookupInteractor lookup = new CoABSLookupInteractor(dep, desc);
Interaction lookInt = new Interaction(dep, lookup, dep.externalManager());
Vector weatherAgents = (Vector) lookInt.start(true);
