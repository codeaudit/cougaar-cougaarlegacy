Choosing an External Agent
•Ideal criteria
–Pre-existing CoABS Grid-ready component
–Some domain overlap or cross-domain impact with ALP (both in function and situation)
–Ability to deal with real-world details (no toys)
–Simple enough to achieve interoperability in a few weeks
–Not another legacy system interface that could be better handled via an ALP LDM PlugIn
